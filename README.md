## Development

```bash
npm install
npm run dev
```

The single `npm run dev` command now builds the project in watch mode and serves the `dist` folder so you can visit the app immediately.

> [!WARNING]
> Unfortunately not all features are done yet. Specifically:
> - Undoing/redoing moves
> - Creating your own sudoku games
